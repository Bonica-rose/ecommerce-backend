# E-Commerce Backend
- A secure and scalable backend that ensures smooth data management and smart
insights

## Features
- Authentication & Authorization
- CRUD Operations
- Product Filtering & Search
- Predictive Analytics (Product Recommendation)
- Security Practices using bcryptjs and jsonwebtoken
- A well-organized folder structure

## Tech Stack
- Node.js
- Express.js
- bcryptjs & jsonwebtoken
- Product recommendation using Hugging face
- Git & GitHub
